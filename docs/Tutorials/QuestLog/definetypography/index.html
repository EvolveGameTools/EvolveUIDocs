Define the Typography Elements

A Typography element is a special container that behaves identically to a Text element. You should define a new Typography element when you want to avoid the need to pass a style parameter manually.

info

Typography elements cannot have a backing class.

Create a new UI and Style files to get started:

  • Right click on UserInterface and go to Create > Evolve > UI File.
  • Name it Typography.
  • Enable Create Style File and Create Files In Directory.
  • Click Create.

You should now see a new subdirectory called Typography with a Typography.ui and Typography.style assets inside of it. Open Typography.ui and add the following code:

typography Text_H1();
typography Text_H2();
typography Text_H3();

That's it! You've just created three new Typography elements that you can style individually via a style file. You will use each throughout this project and will style them in the next section.
