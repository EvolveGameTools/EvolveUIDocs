Set Initial Asset References

warning

Images need to be set

In the last section, you created a new module asset but did not setup any asset references. There are many assets that you‚Äôll add throughout this tutorial, but to start you only need to create a new gradient, image, and header. The module uses a system very similar to Unity's Addressables, in that you define an Asset Id, which is a friendly string that you can reference in stylesheets later.

To create the gradient, click on the QuestLogModule asset and expand the gradients section. Click the + button to add a new row and set the following properties:

  • Asset Id: quest_gradients
  • Is Reflected: true
  • Resolution: 128

Click on the gradient property and set the following four nodes:

  • Node 1: Alpha (200) and Location (0%).
  • Node 2: Alpha (0) and Location (75%).
  • Node 3: Color (#00FFEE) and Location (0%).
  • Node 4: Color (#FFFFFF) and Location (52%).

Add one more gradient with the following properties:

  • Asset Id: scrollbar-gradient
  • Is Reflected: false
  • Resolution: 128
  • Node 1: Alpha (255) and Location (0%).
  • Node 3: Color (#34B0A3) and Location (0%).
  • Node 4: Color (#A4FFF5) and Location (100%).

Next, expand the Fonts section and click the + button to add a new row. Set the following properties:

  1. Asset_id: H1
  2. Font: Bebas Neue Cyrillic_64_UISDF

Repeat the process again to add two more fonts.

  1. Asset_id: H2 and Font: Cabin-Bold_64_UISDF
  2. Asset_id: Normal and Font: Cabin-SemiBold_64_UISDF

Expand the Textures section and click the + button to add a new row. Change the following properties, leaving border at 0:

  • Asset Id: quest_log_background
  • Texture: Background

Add two more Colors, which will be used by the font assets later:

  • Name: PrimaryAccentColor and Color: #77E2C7
  • Name: SecondaryAccentColor and Color: #BACED4
