Create the Main Screen

Open the AppRoot.ui file that you created earlier and add the following code:

//#1
template AppRoot : AppRoot() {
   //#2
   Div(style = [@header-row]) {
      Div(style = [@gradient-line]);
      Text_H1("Quests");
      Div(style = [@gradient-line]);
   }
}

AppRoot.ui will be the entry point for the entire quest log user interface. The code above is doing two things - defining the template file structure and drawing the first bits of UI. More specifically:

Section #1 is the definition of the file, which must follow the format of template FileName : BackingClassName. Every template requires a backing C# class that must extend UIElement. In this example, the backing class came with the starter project and can be found at Assets/Evolve/Scripts/AppRoot.cs.

info

You do not need to create a new backing class for each template. Instead, you can reuse the same backing class across multiple templates. This is useful if you have shared pieces of logic or data that you want to display. There are three ways that you can provide a backing class - read more HERE.

Section #2 is creating the top header of the Quest Log, which starts with a main Div container that is set to the header-row style. That div contains three children:

  • Children 1 and 3 are the same - a div that uses the gradient-line style you defined in QuestLogModule to draw a gradient that goes from transparent to light blue.
  • Child 2 is the Text_H1 typography element that you just created. It has one parameter, which is the string of text that it should display on the UI.

This looks close, but not quite correct! Two things need to be fixed; the first gradient needs to be the opposite direction and the title text would look better if it had some extra spacing.

Fixing the Gradient‚Äã

The first child of header-row that references gradient-line is the only one that needs to be changed, since the last one is facing the correct direction. Fortunately, Evolve has two really handy features to solve this - the ability to nest styles and to query based on the child index.

Update the gradient-line style with the following code:


style gradient-line {
    MarginTop = 12px;
    BackgroundImage = linear-gradient("quest_gradient");
    PreferredSize = 300px 4px;

    [when:first-child] {
        BackgroundImageRotation = 180deg;
    }
}

[when:first-child] is a query that tells Evolve to only apply the defined style properties if the element is, well, the first child. BackgroundImageRotation is a handy style property that lets you rotate the background image by a defined amount, which in this case is flipped 180 degrees.

info

There are a lot of different ways that you can query to determine when a style should be applied. For example, you can apply a style to only the second child with when:child-count(== 2)] or [hover] for when the user is over the element. See {TOPIC NAME} to read more.

Go back to Unity and either press play or if it‚Äôs still running, push control + G to refresh the UI. You should now see the first gradient facing the opposite direction.

Fixing the font spacing‚Äã

It‚Äôs fairly common to define a base style that you want to alter only in certain circumstances. For this tutorial, the Text_H1 style will be built with the flexibility to apply it with or without lettering spacing using a style attribute. An attribute is a way to define a custom key that can be used by the query engine to determine if styling should occur.

Update the Text_H1 style with the following code:

style <Title_H1> {
    TextColor = white;
    TextFontSize = 48px;
    TextTransform = UpperCase;
    TextFontAsset = @H1_Font;

    [attr:characterSpacing] {
        TextCharacterSpacing = 0.2em;
    }
}

Attribute definition is formatted as [attr:yourCustomName] { //styles }. The custom name is what you will reference when defining the element to tell Evolve to apply the style. This means that you will need to update the Text_H1 element in AppRoot to reference the attribute. Head over to the file and replace Text_H1 with the highlighted code below:

template AppRoot : AppRoot() {
   Div(style = [@header-row]) {
      Div(style = [@gradient-line]);
      Text_H1("Quests", attr:characterSpacing);
      Div(style = [@gradient-line]);
   }
}

Go back to Unity and either press play or if it‚Äôs still running, push control + G to refresh the UI. You should now see the first gradient facing the opposite direction.
